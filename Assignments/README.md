### How to Use This Repository

1. **Cloning the Repository**: You can clone this repository to your local machine using the following command:
   ```
   git clone <repository-url>
   ```
2. **Navigating the Structure**: Use the directory structure above to find assignments, lecture notes, and lab exercises.
3. **Completing Assignments**: Each assignment folder contains instructions and resources needed to complete the tasks. Follow the guidelines provided in each assignment.
