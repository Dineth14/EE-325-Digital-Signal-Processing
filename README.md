## EE 325 Digital Signal Processing

This repository contains detailed lecture notes and assignments for the Digital Signal Processing course.

### Assignments

1. Sampling and Aliasing
2. Time Domain Analysis of LTI Systems
3. Discrete Time Convolution
4. Z-Transform
5. System Analysis
6. Realization
7. Stability of Discrete-Time Systems

## How to Run

1. Clone the repository

git clone https://github.com/Dineth14/EE-325-Digital-Signal-Processing.git

2. Open the assignment in MATLAB
3. Run the script
