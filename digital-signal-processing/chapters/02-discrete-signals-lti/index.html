Theory

§ 2.1 Elementary Discrete-Time Signals

Impulse, step, ramp, exponential, complex exponential

A discrete-time signal is a function defined only at integer values of the independent variable $n$. The most fundamental building block is the unit impulse:

$$\delta[n] = \begin{cases} 1, & n=0 \\ 0, & n\neq 0 \end{cases}$$

Every discrete signal can be decomposed as a weighted sum of shifted impulses (the sifting property):

$$x[n] = \sum_{k=-\infty}^{\infty} x[k]\,\delta[n-k]$$

Other essential signals:

  • Unit step: $u[n] = \begin{cases}1,&n\ge0\\0,&n<0\end{cases}$
  • Ramp: $r[n] = n\,u[n]$
  • Real exponential: $x[n] = a^n\,u[n]$, decaying for $|a|<1$
  • Complex exponential: $x[n] = A\,e^{j(\omega_0 n + \phi)}$

The unit step relates to the impulse through the running sum $u[n]=\sum_{k=-\infty}^{n}\delta[k]$, and conversely $\delta[n]=u[n]-u[n-1]$.

§ 2.2 Signal Energy, Power, and Periodicity

Energy signals, power signals, periodicity conditions

The total energy of a discrete signal is:

$$E = \sum_{n=-\infty}^{\infty} |x[n]|^2$$

The average power is:

$$P = \lim_{N\to\infty}\frac{1}{2N+1}\sum_{n=-N}^{N}|x[n]|^2$$

Signals with $0 < E < \infty$ are energy signals (finite energy, zero power). Signals with $0 < P < \infty$ are power signals (infinite energy, finite power). A signal cannot be both.

Periodicity: A discrete signal $x[n]$ is periodic with period $N$ if $x[n+N]=x[n]$ for all $n$. Unlike continuous signals, $x[n]=\cos(\omega_0 n)$ is periodic only when $\omega_0/(2\pi)$ is rational.

$$\frac{\omega_0}{2\pi}=\frac{m}{N} \quad\Rightarrow\quad \text{period } N, \text{ where } \gcd(m,N)=1$$

§ 2.3 Discrete-Time Systems & Properties

Linearity, time-invariance, causality, stability, memory

A system $T\{\cdot\}$ maps an input sequence $x[n]$ to an output sequence $y[n]$. Key properties:

  • Linearity (Superposition): $T\{a\,x_1[n]+b\,x_2[n]\} = a\,T\{x_1[n]\}+b\,T\{x_2[n]\}$
  • Time-Invariance: If $y[n]=T\{x[n]\}$ then $y[n-n_0]=T\{x[n-n_0]\}$
  • Causality: $y[n]$ depends only on $x[k]$ for $k\le n$
  • Stability (BIBO): Bounded input → bounded output
  • Memory: A memoryless system has $y[n]$ depending only on $x[n]$

Linearity is tested by checking both additivity and homogeneity (scaling). A system violating either is non-linear.

§ 2.4 The Convolution Sum

Impulse response, flip-and-slide, convolution properties

For an LTI system, the output is completely determined by its impulse response $h[n]=T\{\delta[n]\}$:

$$y[n] = x[n]*h[n] = \sum_{k=-\infty}^{\infty} x[k]\,h[n-k]$$

The graphical method (flip-and-slide):

  1. Plot $x[k]$ — the input sequence indexed by $k$.
  2. Flip $h[k]$ to get $h[-k]$, then shift right by $n$ to get $h[n-k]$.
  3. For each $n$, multiply $x[k]\cdot h[n-k]$ element-wise and sum.
  4. The sum gives $y[n]$ for that particular $n$.

Convolution properties:

  • Commutative: $x*h = h*x$
  • Associative: $(x*h_1)*h_2 = x*(h_1*h_2)$
  • Distributive: $x*(h_1+h_2) = x*h_1+x*h_2$
  • Identity: $x[n]*\delta[n] = x[n]$

If $x[n]$ has length $L_x$ and $h[n]$ has length $L_h$, then $y[n]=x*h$ has length $L_x+L_h-1$.

§ 2.5 BIBO Stability & Causality

Absolute summability, causal systems, stability conditions

An LTI system is BIBO stable if and only if its impulse response is absolutely summable:

$$\sum_{n=-\infty}^{\infty} |h[n]| < \infty$$

Proof sketch: If the input is bounded ($|x[n]|\le B_x$), then: $$|y[n]| = \left|\sum_k x[k]\,h[n-k]\right| \le B_x \sum_k |h[k]|$$ which is finite when $h[n]$ is absolutely summable.

An LTI system is causal if and only if $h[n]=0$ for $n<0$. Causal + BIBO stable systems have impulse responses that decay to zero.

Example: Consider $h[n]=a^n u[n]$ with $|a|<1$. Then $\sum_{n=0}^{\infty}|a|^n = \frac{1}{1-|a|}< \infty$, so the system is BIBO stable.

Key Equations

Sifting Property

$$x[n]=\sum_k x[k]\,\delta[n-k]$$

Any signal is a weighted sum of shifted impulses. This identity underpins convolution.

Convolution Sum

$$y[n]=\sum_k x[k]\,h[n-k]$$

The output of an LTI system equals the convolution of input and impulse response.

BIBO Stability

$$\sum_{n=-\infty}^{\infty}|h[n]|<\infty$$

Absolute summability of the impulse response guarantees bounded output for bounded input.

Signal Energy

$$E=\sum_{n=-\infty}^{\infty}|x[n]|^2$$

Total energy is the sum of squared magnitudes. Finite energy signals are energy signals.
