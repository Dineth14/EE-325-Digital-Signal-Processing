    The Z-Transform

    Map discrete sequences to the complex z-plane and unlock algebraic tools for system analysis.

    Learning Objectives

    Theory

    The bilateral Z-transform of $x[n]$ is:

    $$X(z)=\sum_{n=-\infty}^{\infty}x[n]\,z^{-n}$$

    where $z=re^{j\omega}$ is a complex variable. The region of convergence (ROC) is the set of $z$ values for which the sum converges absolutely:

    $$\text{ROC}: \{z : \sum_{n=-\infty}^{\infty}|x[n]|\,|z|^{-n}<\infty\}$$

    The ROC is always an annular region $R_1<|z|Key facts:

    • The ROC never contains poles of $X(z)$.
    • For a right-sided (causal) signal: ROC extends outward from the outermost pole.
    • For a left-sided (anti-causal) signal: ROC extends inward from the innermost pole.
    • For a two-sided signal: ROC is a ring between two pole radii.

    Unilateral Z-transform: restricts summation to $n\ge0$, useful for solving difference equations with initial conditions.

    Fundamental pairs every DSP engineer should memorize:

    $$\delta[n] \;\longleftrightarrow\; 1 \quad\text{(ROC: all } z\text{)}$$
    $$a^n u[n] \;\longleftrightarrow\; \frac{1}{1-az^{-1}} \quad\text{(ROC: }|z|>|a|\text{)}$$
    $$n\,a^n u[n] \;\longleftrightarrow\; \frac{az^{-1}}{(1-az^{-1})^2} \quad\text{(ROC: }|z|>|a|\text{)}$$
    $$u[n] \;\longleftrightarrow\; \frac{1}{1-z^{-1}} \quad\text{(ROC: }|z|>1\text{)}$$

    For a damped sinusoid $r^n\cos(\omega_0 n)u[n]$:

    $$\frac{1-r\cos(\omega_0)z^{-1}}{1-2r\cos(\omega_0)z^{-1}+r^2z^{-2}}$$

    Linearity: $ax_1[n]+bx_2[n] \;\longleftrightarrow\; aX_1(z)+bX_2(z)$

    Time shifting: $x[n-n_0] \;\longleftrightarrow\; z^{-n_0}X(z)$ (ROC unchanged except possibly $z=0$ or $z=\infty$)

    Convolution: $x_1[n]*x_2[n] \;\longleftrightarrow\; X_1(z)\cdot X_2(z)$ (ROC ⊇ intersection of individual ROCs)

    Differentiation in z: $n\cdot x[n] \;\longleftrightarrow\; -z\frac{dX(z)}{dz}$

    Initial value theorem: $x[0]=\lim_{z\to\infty}X(z)$ (for causal signals)

    Final value theorem: $\lim_{n\to\infty}x[n]=\lim_{z\to1}(1-z^{-1})X(z)$ (if the limit exists)

    For an LTI system described by $\sum_{k=0}^{N}a_k y[n-k]=\sum_{k=0}^{M}b_k x[n-k]$, taking the Z-transform yields:

    $$H(z)=\frac{Y(z)}{X(z)}=\frac{\sum_{k=0}^{M}b_k z^{-k}}{\sum_{k=0}^{N}a_k z^{-k}}=\frac{b_0\prod_{k=1}^{M}(1-c_k z^{-1})}{a_0\prod_{k=1}^{N}(1-d_k z^{-1})}$$

    where $c_k$ are zeros and $d_k$ are poles. Key relationships:

    • Stability: System is BIBO stable iff the ROC includes the unit circle $|z|=1$; for causal systems this requires all poles inside the unit circle.
    • Frequency response: $H(e^{j\omega})=H(z)|_{z=e^{j\omega}}$ — evaluate $H(z)$ on the unit circle.
    • Poles near the unit circle cause peaks in $|H(e^{j\omega})|$; zeros cause notches.

    Three principal methods for computing $x[n]=\mathcal{Z}^{-1}\{X(z)\}$:

    1. Partial Fraction Expansion: Decompose $X(z)$ into simple fractions and use known Z-transform pairs:

    $$X(z)=\sum_{k=1}^{N}\frac{A_k}{1-d_k z^{-1}} \;\;\Rightarrow\;\; x[n]=\sum_{k=1}^{N}A_k\,d_k^n\,u[n]$$

    (for the causal ROC). For anti-causal ROC, each term becomes $-A_k\,d_k^n\,u[-n-1]$.

    2. Long Division: Divide numerator by denominator to obtain coefficients of $z^{-n}$ directly: $X(z)=x[0]+x[1]z^{-1}+x[2]z^{-2}+\cdots$

    3. Contour Integration: $x[n]=\frac{1}{2\pi j}\oint X(z)z^{n-1}dz$ — typically evaluated via the residue theorem.

    Key Equations

    Z-Transform Definition

    $$X(z)=\sum_{n=-\infty}^{\infty}x[n]z^{-n}$$

    Maps a discrete sequence to a function of the complex variable $z$. Convergence requires the ROC.

    Transfer Function

    $$H(z)=\frac{B(z)}{A(z)}=\frac{\sum b_k z^{-k}}{\sum a_k z^{-k}}$$

    Ratio of output to input Z-transforms. Poles of $H(z)$ determine stability and natural modes.

    Convolution ↔ Multiplication

    $$x*h \;\longleftrightarrow\; X(z)H(z)$$

    Convolution in time becomes multiplication in the Z-domain — the key payoff of the transform.

    Causal Stability

    $$\text{All poles } |d_k| < 1$$

    For a causal LTI system, BIBO stability requires every pole to lie strictly inside the unit circle.
