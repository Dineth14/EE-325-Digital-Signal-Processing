FIR Filter Design

Design finite impulse response filters using windowed-sinc methods and understand linear phase.

Learning Objectives

  • Design FIR lowpass, highpass, and bandpass filters using the windowed-sinc method.
  • Compare window functions and their impact on sidelobe levels and transition width.
  • Classify FIR filters into the four linear-phase types and state their constraints.
  • Relate filter order to transition bandwidth and stopband attenuation.

Theory

▶

§5.1 Ideal vs. Realizable FIR Filters

Sinc impulse response, truncation requirements

An ideal lowpass filter has frequency response $H_d(e^{j\omega})=\begin{cases}1,&|\omega|\le\omega_c\\0,&\omega_c<|\omega|\le\pi\end{cases}$.

Its impulse response is the sinc function:

$$h_d[n]=\frac{\sin(\omega_c n)}{\pi n}, \quad -\infty < n < \infty$$

This is noncausal and infinite-length — not realizable. We truncate and window it to create a practical FIR filter.

▶

§5.2 Windowed-Sinc Design Method

Three-step design procedure

The design procedure:

  1. Compute the ideal impulse response $h_d[n]$ for the desired frequency response.
  2. Truncate to $N$ samples: $h_t[n]=h_d[n]$ for $0\le n\le N-1$.
  3. Apply a window: $h[n]=h_t[n]\cdot w[n]$.
$$h[n]=\frac{\sin[\omega_c(n-M)]}{\pi(n-M)}\cdot w[n], \quad M=\frac{N-1}{2}$$

The window controls the tradeoff between main-lobe width (transition band) and sidelobe level (stopband attenuation).

§5.4 Linear Phase FIR Filters

Four types and symmetry constraints

An FIR filter has generalized linear phase if $H(e^{j\omega})=A(\omega)\,e^{-j(\alpha\omega+\beta)}$, where $A(\omega)$ is a real amplitude function. This requires symmetry of coefficients:

Four types:

  • Type I: $h[n]=h[N-1-n]$ (symmetric), $N$ odd. All frequency-selective filters possible.
  • Type II: $h[n]=h[N-1-n]$ (symmetric), $N$ even. Cannot implement highpass ($H(e^{j\pi})=0$).
  • Type III: $h[n]=-h[N-1-n]$ (anti-symmetric), $N$ odd. Differentiators, Hilbert transforms. $H(e^{j0})=H(e^{j\pi})=0$.
  • Type IV: $h[n]=-h[N-1-n]$ (anti-symmetric), $N$ even. Differentiators. $H(e^{j0})=0$.

Linear phase ⇒ constant group delay = $(N-1)/2$ samples. This preserves the shape of signals passing through the filter.

▶

§5.5 Kaiser Window Design Formula

Empirical order and β formulas

Kaiser's empirical formulas relate filter specifications to window parameters:

Given stopband attenuation $A_s=-20\log_{10}\delta_s$ (dB) and transition width $\Delta\omega$:

$$\beta=\begin{cases}0.1102(A_s-8.7),&A_s>50\\0.5842(A_s-21)^{0.4}+0.07886(A_s-21),&21\le A_s\le50\\0,&A_s<21\end{cases}$$
$$N=\lceil\frac{A_s-7.95}{2.285\,\Delta\omega}\rceil+1$$

Key Equations

Windowed Sinc

$$h[n]=\frac{\sin[\omega_c(n-M)]}{\pi(n-M)}\cdot w[n]$$

Fundamental FIR design formula. The window shapes the tradeoff between transition width and stopband attenuation.

Group Delay

$$\tau_g = \frac{N-1}{2}\text{ samples}$$

Constant group delay for all linear-phase FIR filters. Signal shape is preserved.

Kaiser Order

$$N=\lceil\frac{A_s-7.95}{2.285\,\Delta\omega}\rceil+1$$

Empirical formula relating stopband attenuation and transition width to filter order.

Linear Phase Condition

$$h[n]=\pm h[N\!-\!1\!-\!n]$$

Symmetric (+) or anti-symmetric (−) impulse response guarantees linear phase.

The Blackman window achieves the highest stopband attenuation (74 dB) among common fixed windows, at the cost of a wider main lobe (transition band).

Type II filters have $H(e^{j\pi})=0$ due to their symmetry and even length, making them unable to realize a highpass response that requires gain at $\omega=\pi$.

The transition band width is inversely proportional to the filter order. The stopband attenuation is determined primarily by the window type, not the order.
