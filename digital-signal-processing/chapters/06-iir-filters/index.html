    IIR Filter Design

    Design recursive filters from analog prototypes using the bilinear transform, and explore stability.

    Learning Objectives

    Theory

    An IIR (Infinite Impulse Response) filter has a transfer function with both poles and zeros:

    $$H(z)=\frac{\sum_{k=0}^{M}b_k z^{-k}}{1+\sum_{k=1}^{N}a_k z^{-k}}$$

    Because of the feedback (denominator terms), IIR filters can achieve sharp frequency selectivity with far fewer coefficients than FIR filters.

    PropertyFIRIIR
    Impulse responseFinite lengthInfinite (decaying)
    Linear phaseGuaranteed (symmetric)Not possible in general
    StabilityAlways stableMust check: all poles inside UC
    EfficiencyHigher order neededLower order for same spec
    Design methodWindowed sinc, Parks-McClellanAnalog prototype + transform

    The IIR approach is particularly effective for standard lowpass/highpass/bandpass specifications where decades of analog filter theory (Butterworth, Chebyshev, Elliptic) can be leveraged.

    The IIR design pipeline begins with a normalized analog lowpass prototype $H_a(s)$ with cutoff frequency $\Omega_c = 1$ rad/s.

    Butterworth Filter

    The Butterworth filter provides maximally flat magnitude in the passband. The squared magnitude:

    $$|H_a(j\Omega)|^2=\frac{1}{1+\left(\frac{\Omega}{\Omega_c}\right)^{2N}}$$

    The poles of $H_a(s)H_a(-s)$ lie on a circle of radius $\Omega_c$ in the $s$-plane. We keep the left-half-plane poles for stability:

    $$s_k=\Omega_c\, e^{j\pi(2k+N+1)/(2N)},\quad k=0,1,\dots,N-1$$

    Chebyshev Type I

    Chebyshev I filters have equiripple passband and monotonic stopband, achieving a sharper transition for the same order:

    $$|H_a(j\Omega)|^2=\frac{1}{1+\epsilon^2\, T_N^2\!\left(\frac{\Omega}{\Omega_c}\right)}$$

    where $T_N$ is the $N$-th order Chebyshev polynomial of the first kind, and $\epsilon$ controls passband ripple: $R_p = 10\log_{10}(1+\epsilon^2)$ dB.

    Chebyshev Type II (Inverse)

    Has a flat passband but equiripple stopband — useful when passband flatness is critical.

    Elliptic (Cauer)

    Equiripple in both passband and stopband — gives the absolute minimum order for given specs, at the cost of the most complex design equations.

    The bilinear transform maps the entire $s$-plane to the $z$-plane via:

    $$s = \frac{2}{T}\cdot\frac{1-z^{-1}}{1+z^{-1}} \quad\Longleftrightarrow\quad z = \frac{1+\frac{sT}{2}}{1-\frac{sT}{2}}$$

    Key mapping properties:

    • The $j\Omega$ axis maps to the unit circle $|z|=1$.
    • The left half-plane ($\text{Re}(s)<0$) maps to $|z|<1$ — stability is preserved.
    • The right half-plane maps to $|z|>1$.

    Frequency Warping

    The bilinear transform introduces a nonlinear compression of the frequency axis:

    $$\Omega = \frac{2}{T}\tan\left(\frac{\omega}{2}\right) \quad\Longleftrightarrow\quad \omega = 2\arctan\left(\frac{\Omega T}{2}\right)$$

    To ensure the digital filter's critical frequency matches the specification, we pre-warp:

    $$\Omega_c = \frac{2}{T}\tan\left(\frac{\omega_c}{2}\right)$$

    This pre-warped $\Omega_c$ is used as the cutoff when designing the analog prototype.

    IIR filters can be realized using several standard structures. The general difference equation:

    $$y[n]=\sum_{k=0}^{M}b_k\, x[n-k]-\sum_{k=1}^{N}a_k\, y[n-k]$$

    Direct Form I

    Implements the numerator and denominator as two separate FIR sections in cascade. Requires $M+N$ delay elements.

    Direct Form II

    Shares a single delay line for both numerator and denominator — requires only $\max(M,N)$ delay elements. Also called the canonical form.

    Cascade (Second-Order Sections)

    Factors $H(z)$ into a product of biquad sections:

    $$H(z)=\prod_{k}\frac{b_{0k}+b_{1k}z^{-1}+b_{2k}z^{-2}}{1+a_{1k}z^{-1}+a_{2k}z^{-2}}$$

    This is the preferred implementation in practice because it minimizes coefficient sensitivity and finite-precision effects.

    Parallel Form

    Decomposes $H(z)$ using partial fractions into a sum of second-order sections.

    A causal LTI system is BIBO stable if and only if all poles of $H(z)$ lie strictly inside the unit circle:

    $$|p_k| < 1, \quad \forall\, k$$

    Stability Tests

    • Pole inspection: Factor the denominator and check all root magnitudes.
    • Jury test: An algebraic test on the denominator coefficients (analogous to the Routh test for continuous systems).
    • Schur-Cohn test: Uses a recursion on reflection coefficients.

    Marginal Stability

    If poles lie on the unit circle ($|p_k|=1$) with multiplicity 1, the system is marginally stable — bounded input may produce unbounded output. If any repeated pole is on the unit circle, the system is unstable.

    Effect of Coefficient Quantization

    In fixed-point implementations, small changes in coefficients can shift poles outside the unit circle, causing instability. Cascade biquad forms are less sensitive than direct forms.

    Key Equations

    Bilinear Transform

    $$s=\frac{2}{T}\,\frac{1-z^{-1}}{1+z^{-1}}$$

    Maps the entire s-plane to the z-plane. Preserves stability: LHP → inside UC, jω axis → UC.

    Frequency Pre-Warping

    $$\Omega_c=\frac{2}{T}\tan\!\left(\frac{\omega_c}{2}\right)$$

    Corrects for the nonlinear frequency compression of the bilinear transform. Critical for matching specs.

    Butterworth Magnitude

    $$|H(j\Omega)|^2=\frac{1}{1+(\Omega/\Omega_c)^{2N}}$$

    Maximally flat magnitude response. No ripple in passband or stopband. Roll-off is 20N dB/decade.

    BIBO Stability

    $$|p_k| < 1 \;\;\forall\, k$$

    All poles of H(z) must be strictly inside the unit circle for a causal system to be BIBO stable.
